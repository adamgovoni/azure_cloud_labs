# azure_cloud_labs
Hands-on Azure learning labs covering networking, virtual machines, identity, security, and monitoring while studying for the AZ-104 Azure Administrator certification.
The goal of these labs is to understand how core Azure services work by building small environments and documenting the architecture and lessons learned.

Topics covered include:

- Virtual networks and subnet design
- Network security groups and access control
- Virtual machine deployment
- Azure identity and RBAC
- Storage services
- Monitoring and logging

Each lab folder contains:
- architecture diagrams
- notes on configuration
- lessons learned
